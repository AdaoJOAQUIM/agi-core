> ## ðŸ“¦ Core Modules
>
> ### 1. Executive Controller
> The central orchestrator implementing the **Global Workspace Theory**.
> - `orchestrator.py` - Main control loop (LangGraph)
> - - `goal_manager.py` - Goal decomposition (AutoGen)
>   - - `scheduler.py` - Task scheduling (CrewAI)
>    
>     - ### 2. Memory System
>     - Multi-tier memory architecture inspired by human cognition.
>     - - `working_memory.py` - Short-term buffer (Letta/MemGPT)
>       - - `episodic_memory.py` - Experience storage (ChromaDB)
>         - - `semantic_memory.py` - Knowledge base (LlamaIndex)
>           - - `procedural_memory.py` - Skills & procedures
>            
>             - ### 3. Reasoning Engine
>             - Advanced deliberative reasoning capabilities.
>             - - `chain_of_thought.py` - Linear reasoning (Wei et al. 2022)
>               - - `tree_of_thoughts.py` - Tree search (Yao et al. 2023)
>                 - - `reflexion.py` - Self-reflection (Shinn et al. 2023)
>                   - - `mcts.py` - Monte Carlo Tree Search
>                    
>                     - ### 4. Tools Engine
>                     - Function calling and external interaction.
>                     - - `function_caller.py` - API execution (Gorilla)
>                       - - `code_executor.py` - Safe code execution (E2B/Modal)
>                         - - `web_browser.py` - Web interaction (Playwright)
>                          
>                           - ### 5. Perception Layer
>                           - Multi-modal input processing.
>                           - - `text.py` - NLP & embedding (OpenAI/Anthropic)
>                             - - `vision.py` - Image understanding (GPT-4V/Claude)
>                               - - `audio.py` - Speech & audio (Whisper)
